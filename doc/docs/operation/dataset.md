## 项目简介
- 项目源代码地址: <https://gitee.com/anji-plus/report>
- 在线提问: <https://gitee.com/anji-plus/report/issues>
- 在线文档: 

大屏设计（Report）是一个可视化拖拽编辑的，直观，酷炫，具有科技感的图表工具项目。
内置的基础功能包括数据源，数据集，国际化，字典管理，报表管理


## 技术选型

**1.环境**
-  JavaEE 8
-  Apache Maven 3

**2.主框架**
-  Spring Boot 2.3.x

**3.持久层**
-  Apache MyBatis 3.5.x

**4.视图层**
-  Vue 2.6.10+
-  ElementUI 2.13.0+


## 系统特性
1. 最新最稳定的技术栈；
2. 支持多数据源配置
3. 丰富的大屏组件。拖拽配置实现动态大屏
