## 打包目录
```
├── bin                                           启动命令脚本
│   ├── logs                                      启动日志目录
│   ├── cache                                     本地缓存目录
│   ├── startup.cmd
│   ├── shutdown.cmd
│   ├── startup.sh
│   └── shutdown.sh
├── config                                        配置文件目录
├── lib                                           自定义扩展包
├── target                                        report-core核心包
```

## 系统目录
```
├── doc                                           文档源码
│   ├── docs
│   ├── package.json
│   └── README.md
├── pom.xml                                       父pom，jar版本管理
├── report-core                                   java源码
│   ├── pom.xml                                   gaea父pom，jar版本管理
│   └── README.md

├── LICENSE

```
